# GEM

GEM is an interactive **Dash application** that allows users to explore
**rates of return** and visualize charts for selected ETFs.
It provides an intuitive interface for investors and analysts to monitor
performance and make data-driven decisions.

---

## Features

- Select one or more ETFs to analyze.
- Select any date range to analyze.
- View historical rates of return.
- Interactive charts for returns.
